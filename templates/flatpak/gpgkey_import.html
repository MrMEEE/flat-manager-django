How to Export Keys

To export keys from an existing GPG keyring:

# Export public key
gpg --armor --export KEY_ID

# Export private key
gpg --armor --export-secret-key KEY_ID

Security: If your private key is encrypted, the passphrase will be used to decrypt it during import. The decrypted key is stored in the database (passphrases are not stored).
